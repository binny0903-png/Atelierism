# Atelierism
디자이너와 고객을 이어주는 인테리어 매칭 플랫폼입니다.
사용자는 원하는 디자이너를 찾고, 인테리어 상담 및 결제를 진행할 수 있으며
디자이너는 본인의 포트폴리오를 등록하고 고객과 소통할 수 있습니다.

# 개발 목적
인테리어 디자이너와 고객을 연결하는 플랫폼 구축
고객이 인테리어 상담/신청/결제를 한 곳에서 처리할 수 있는 환경 제공
디자이너는 본인의 작업물을 공유하고 고객을 유치할 수 있는 시스템 제공

# 사용 기술
Front-end : react
back-end : Spring
DataBase : SqlDeveloper(Oracle)
협업 : github

# 담당 기능 및 페이지
1. 로그인
    1-1. JWT 기반 로그인 기능 구현
    1-2. 비밀번호 암호화 저장
2. 회원가입
    2-1. 유효성 검사
    2-2. 아이디 중복 체크
    2-3. 이메일 인증코드 전송 기능
    2-4. 우편번호 조회 API
3. 일반회원 마이페이지
    3-1. 일반회원 정보 수정 : 마이페이지에 보여지는 정보들 수정 기능
    3-2. 회원 탈퇴 : 완전 삭제 처리
    3-3. 결제 내역 : 고객 결제 이력 불러오기
4. 아이디 찾기 : 이름 + 이메일 기반 계정 찾기
5. 비밀번호 찾기 : 본인 인증 후 비밀번호 재설정 기능 제공
6. 사이드 메뉴의 프로필 등록 기능 : 프로필 사진 업로드 기능 제공
